## 三、添加路由配置
在`src/router/index.ts`中添加学习管理页面的路由：

```typescript
{
  path: '/campus-life/study-management',
  name: 'study-management',
  component: () => import('../views/StudyManagement.vue'),
}
```

## 四、实现样式和响应式设计
1. 顶部标题栏样式：
   - 高度60px，背景色#409eff
   - 文字居中，白色，20px加粗

2. 左侧导航栏样式：
   - 宽度200px，固定定位，背景#f5f7fa
   - 菜单项高度48px
   - hover背景#e6f7ff
   - 选中项背景#1890ff，文字白色

3. 右侧内容区样式：
   - 占剩余宽度，内边距20px
   - 卡片上下排列，间距20px
   - 卡片hover阴影：0 4px 12px rgba(0,0,0,0.1)
   - 图表容器宽100%，高300px

4. 响应式设计：
   - 屏幕<768px时，左侧导航栏折叠为图标模式
   - 右侧内容区占满宽度
   - 卡片100%宽度堆叠

## 六、添加ECharts初始化代码
在组件mounted钩子中初始化图表容器：

```typescript
import * as echarts from 'echarts'

onMounted(() => {
  // 初始化统计图表
  const statisticsChart = echarts.init(document.getElementById('statistics-chart'))
  // 初始化分析图表
  const analysisChart = echarts.init(document.getElementById('analysis-chart'))
  
  // 设置图表配置（占位）
  const option = {
    title: { text: '统计图表' },
    tooltip: {},
    xAxis: { type: 'category', data: [] },
    yAxis: { type: 'value' },
    series: [{ data: [], type: 'bar' }]
  }
  
  statisticsChart.setOption(option)
  analysisChart.setOption(option)
})
```
